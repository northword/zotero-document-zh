本仓库存储 Zotero 中文社区所建立的各种文档，包括 Zotero 用户文档、Zotero 插件开发文档、CSL 规范中文译文等。

## 访问

请访问在线网站：[Zotero 中文社区 (https://zotero-chinese.com)](https://zotero-chinese.com)。

备用地址：<https://zotero-chinese.github.io>。
